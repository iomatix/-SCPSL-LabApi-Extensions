# LabAPI Extensions

A high-performance, allocation-optimized utility and fluent API extension framework for LabAPI (SCP: Secret Laboratory modding engine).

## 🚀 Key Features
- **Zero-Allocation Timing Utilities**: Fluent wrappers over MEC coroutines.
- **Advanced Spatial Queries**: Sub-frame positioning and unified entity distance maps.
- **Native Transform Tracking**: Native Unity anchor components for smooth target following.
